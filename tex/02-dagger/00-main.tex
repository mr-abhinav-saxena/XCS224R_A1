\section{DAgger (Dataset Aggregation)}

In standard Behavior Cloning, the agent is only trained on states the expert visited, so at test time it may drift into unfamiliar states and fail. DAgger changes this by letting the agent interact with the world, collecting the states the agent actually visits, including its mistakes.
